## 🐋 프로젝트 개요

- 판매자는 플랫폼을 통해 파트너쉽을 맺은 후 레저 업체를 등록하고, 사용자는 제주도의 레저를 편리하게 예약할 수 있는 서비스를 제공합니다.
- 카카오 지도를 통해 제주도 레저 업체의 위치를 한눈에 파악할 수 있습니다.
- 스노쿨링/다이빙, 수상레저, 서핑, 승마, ATV로 분류하여 레저별로 검색할 수 있습니다.
- 관심순/평점순/가격순/리뷰순으로 필터링한 검색 결과를 손쉽게 확인할 수 있습니다.
- 토스 페이먼츠를 연동하여 편리하게 결제할 수 있습니다.
- 레저 업체를 이용한 회원들은 리뷰를 남길 수 있고, 다른 고객들이 해당 리뷰들을 참고할 수 있도록 설계했습니다.
